# Successful User Registration

**Priority**: Critical.

## Preconditions:
- Email is not registered.

## Steps:
1.	Navigate to /login.
2.	Switch to “Register” mode.
3.	Enter valid email.
4.	Enter valid password.
5.	Submit form.

## Expected Result:
- User is successfully registered.
- Success message is displayed.
- User is redirected to login form.
