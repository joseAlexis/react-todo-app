# React Todo App
A modern, scalable Todo application built with a monorepo architecture, focused on clean code, automation, and best practices. This project demonstrates a production-ready frontend setup with linting, formatting, pre-commit hooks, and testability in mind.

## 🚀 Features
- ✅ Create, edit, delete todos
- 📌 Mark todos as completed / pending
- 👀 Hide/show completed tasks (enabled by default)
- 🔐 Authentication
- 🧪 Test-friendly structure with data-testid support
- 💾 Persistent state (Json-server atm)

## 📌 Future Improvements
- 🔄 Add workspaces
- 🎒 Add proper API project into monorepo
- 🔍 Filter todos (All / Active / Completed)


## 🛠️ Tech Stack
- React – UI library
- TypeScript – Static typing
- Vite

##  Code Quality
- ESLint – Linting
